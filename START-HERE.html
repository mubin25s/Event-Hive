ğŸ¯ Platform Concept

Event-Hive works like FoodPanda for events:

  • Vendors (community centers, resorts) list their venues and mark availability
  • Clients browse the marketplace, compare options, and book venues
  • Guests receive links to view event details (schedule, menu, theme)

ğŸ”‘ Key Features

For Vendors:

  • Interactive calendar to mark dates as Available/Booked
  • View bookings and revenue statistics
  • Two vendor types: Community Centers (all-inclusive) & Resorts (venue-only)

For Clients:

  • Filter venues by budget ($5k-$10k, etc.), date, and event type
  • See real-time availability
  • Manage guest lists and share event details

For Guests:

  • View event schedule and timeline
  • See menu options and dietary information
  • RSVP functionality
